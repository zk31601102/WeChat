# WeChat 

>- 实现了微信的主界面
>- 通讯录进行了汉语拼音排序
>- 调用[图灵机器人](http://www.tuling123.com/help/h_cent_iossdk.jhtml?nav=doc)来实现聊天
>- 使用CoreData存储聊天记录
>- 完成在**iOS12, Xcode10** 真机测试 
>- 若对您有帮助，不妨点个star吧！
